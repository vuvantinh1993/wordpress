WooCommerce Wishlist is a Plugin to add Customizable Wishlist to your site.

== Description ==

Looking for some cool design to make a wish and add it into a list.

Here is the perfect solution for that. With WooCommerce Wishlist you can save as many as product you like to your Wishtlist with just one click and you can view each product added instantly on a fixed Wishlist box.

Just like this you can save your list for future buy. Each user can have there own wishlist in which they can store there favorite products and access them any time they like. Of course you will need to login before you can access to your Wishlist.

WooCommerce Wishlist Features :

*   Options to let your customers create there own wishlist.	
*   Add shortcode on any page to display the wishlist.
*   Change wishlist icon shape and color as your requirment.
*   Options to postion the add to wishlist icon on each product
*   Options Change the wishlist icon to text.
*	Options to edit title of your wishlist.

Added to all we have an additional feature which display a fixed box with the list of all added products. You can position the box to 6 different location in your site.

== Installation ==

1. Upload "Woo WishList" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Installation complete
